# 🌫️ Misty Auth — Sign Up Page

A modern and elegant sign-up page built with **HTML**, **CSS**, and **JavaScript**, featuring a glassmorphism design, a nature photography background, and a smart auto-hiding navbar.

## 🌟 Overview

This project was built to practice real-world UI design techniques beyond basic HTML and CSS.  
The goal was to create a clean, atmospheric authentication page that feels modern and polished.

It features:
- A **glassmorphism** design with frosted blur effects
- A **smart navbar** that hides when scrolling down and reappears when scrolling up
- A **multi-section layout** with a profile section below the sign-up form
- Smooth **hover animations** on navigation links and buttons

---

## ✨ Main Features

- Glassmorphism card with backdrop blur
- Auto-hiding navbar on scroll (JavaScript)
- Smooth underline hover animation on nav links
- Register button with scale hover effect
- Frosted glass input fields
- Social media icons section
- Fixed full-screen background image
- Second section with profile picture display

---

## 🛠 Built With

- HTML5
- CSS3 (glassmorphism, transitions, animations)
- JavaScript (scroll event handling)
